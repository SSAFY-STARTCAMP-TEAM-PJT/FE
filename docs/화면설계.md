# LocalHub 최종 UI 설계

## 1. 서비스 개요

**LocalHub**는 제공된 여행지 데이터를 지도에서 탐색하고, 여러 여행지를 하나의 코스로 구성하여 익명 게시글로 공유하는 지역 여행 커뮤니티입니다.

RFP의 필수 범위인 익명 게시판 CRUD, SQLite 저장, JSON 여행지 연동, 챗봇, Vue 3·FastAPI 연동 및 Netlify·Render 배포를 기본으로 구성하고, 선택 기능으로 지도 시각화와 여행 코스 공유 기능을 결합합니다.

### 핵심 사용자 흐름

```
여행지 탐색
→ 여행지 상세 확인
→ 여러 여행지 선택
→ 선택 순서대로 코스 구성
→ 게시글 작성 및 공유
→ 게시글 상세에서 코스 지도 확인
→ 여행지별 관련 게시글과 근처 여행지 탐색
```

# 2. 정보 구조

```
LocalHub
├─ 홈
├─ 게시판
│  ├─ 게시글 목록
│  ├─ 게시글 작성
│  ├─ 게시글 상세
│  └─ 게시글 수정
├─ 여행지 지도
│  ├─ 현재 지도 범위 여행지 목록
│  └─ 여행지 상세 패널
└─ 공통 챗봇 위젯
```

## 최종 라우팅

| 경로                  | 화면        | 주요 기능                                 |
| --------------------- | ----------- | ----------------------------------------- |
| `/`                   | 홈          | 서비스 소개, 최근 게시글, 주요 기능 진입  |
| `/posts`              | 게시글 목록 | 검색, 카테고리 필터, 게시글 목록          |
| `/posts/new`          | 게시글 작성 | 익명 글 작성, 여행지 다중 첨부, 코스 구성 |
| `/posts/:postId`      | 게시글 상세 | 본문, 첨부 코스 지도, 여행지 정보         |
| `/posts/:postId/edit` | 게시글 수정 | 비밀번호 확인 후 내용과 코스 수정         |
| `/map`                | 여행지 지도 | 지도 탐색, 검색, 카테고리, 상세 정보      |
| `/:pathMatch(.*)*`    | 404         | 잘못된 주소 안내                          |

## 지도 화면 쿼리 파라미터

| 예시                    | 의미                                    |
| ----------------------- | --------------------------------------- |
| `/map?placeId=15`       | 특정 여행지 상세 표시                   |
| `/map?category=tour`    | 관광지 카테고리만 표시                  |
| `/map?query=해운대`     | 해운대 검색 결과 표시                   |
| `/posts?placeId=15`     | 해당 여행지가 포함된 게시글 조회        |
| `/posts/new?placeId=15` | 해당 여행지를 첨부한 상태로 글쓰기 시작 |

# 4. 홈 화면

## 목적

서비스의 기능을 짧게 소개하고 사용자를 여행지 지도 또는 코스 작성으로 유도합니다.

```
┌──────────────────────────────────────────────┐
│ 선택 권역 소개                              │
│ 여행지를 연결해 나만의 코스를 공유하세요    │
│ [여행지 둘러보기] [코스 공유하기]           │
├──────────────────────────────────────────────┤
│ 최근 여행 코스 게시글                       │
│ [게시글 카드] [게시글 카드] [게시글 카드]    │
├──────────────────────────────────────────────┤
│ 추천 여행지 또는 지도 미리보기              │
└──────────────────────────────────────────────┘
```

## 주요 구성

| 영역          | 내용                                 |
| ------------- | ------------------------------------ |
| 소개 배너     | 서비스 설명과 선택 권역 소개         |
| 주요 버튼     | 여행지 지도, 게시글 작성             |
| 최근 게시글   | 최신 게시글 3~5개                    |
| 지도 미리보기 | 일부 여행지 핀 표시 또는 대표 이미지 |
| 챗봇          | 모든 화면 공통 플로팅                |

# 5. 게시글 목록 화면

**경로:** `/posts`

## 화면 구조

```
┌────────────────────────────────────────────────────┐
│ 게시판                                  [글쓰기]  │
├────────────────────────────────────────────────────┤
│ [검색어 입력________________] [검색] [초기화]      │
├────────────────────────────────────────────────────┤
│ 번호 │ 제목                               │ 작성일 │
│ 12   │ 부산 바다 따라 하루 여행  [📍 +3] │ 07.14 │
│ 11   │ 비 오는 날 실내 코스      [📍 +4] │ 07.14 │
│ 10   │ 해운대 주변 맛집 추천     [📍 +1] │ 07.13 │
├────────────────────────────────────────────────────┤
│                   < 1 2 3 >                        │
└────────────────────────────────────────────────────┘
```

## 장소 첨부 배지

`첨부 장소 수`를 별도 컬럼으로 표시하지 않고 제목 옆에 작은 배지로 표시합니다.

| 첨부 상태   | 표시                     |
| ----------- | ------------------------ |
| 장소 없음   | 배지 미표시              |
| 장소 1개    | 핀 아이콘 `+1`           |
| 장소 3개    | 핀 아이콘 `+3`           |
| 마우스 오버 | 첨부 장소 이름 일부 표시 |

```
부산 바다 따라 하루 여행  [핀 아이콘 +3]
```

제목이 긴 경우 제목은 말줄임 처리하고 배지는 오른쪽에 고정합니다.

## 주요 기능

- 게시글 목록 조회
- 제목·내용 검색
- 게시글 카테고리 필터
- 특정 여행지가 포함된 게시글 필터
- 페이지네이션
- 장소 첨부 수 배지
- 글쓰기 이동

# 6. 게시글 작성 화면

**경로:** `/posts/new`

## 설계 원칙

일반적인 게시글 작성 작업과 여행 코스 구성 작업을 서로 다른 패널로 분리합니다.

## 데스크톱 구조

```
┌──────────────────────────────────────────────────────────────┐
│ 여행 코스 공유하기                                          │
├──────────────────────────────┬───────────────────────────────┤
│ 게시글 작성                 │ 여행지 첨부                  │
│                              │                               │
│ 카테고리 [여행 코스 ▼]      │ [여행지 검색____________]    │
│                              │ [전체][관광지][맛집][문화]   │
│ 제목                         │                               │
│ [_______________________]    │ 검색 결과                    │
│                              │ 해운대 해수욕장         [+]  │
│ 내용                         │ 동백섬                  [+]  │
│ [                       ]    │ 광안리 해수욕장         [+]  │
│ [                       ]    │                               │
│ [                       ]    │ 선택한 여행지                │
│                              │ 1. 해운대       [↑][↓][×]   │
│ 수정용 비밀번호             │ 2. 동백섬        [↑][↓][×]   │
│ [_______________________]    │ 3. 광안리        [↑][↓][×]   │
│                              │                               │
│                              │ 코스 미리보기 지도           │
│                              │ ① ───── ② ───── ③          │
├──────────────────────────────┴───────────────────────────────┤
│                                       [취소] [게시글 등록]   │
└──────────────────────────────────────────────────────────────┘
```

## 왼쪽: 게시글 작성 패널

- 게시글 카테고리
- 제목
- 본문
- 수정·삭제용 비밀번호

## 오른쪽: 여행지 첨부 패널

- 여행지 이름 검색
- 여행지 카테고리 필터
- 검색 결과 목록
- 여행지 추가·제거
- 첨부 순서 변경
- 코스 지도 미리보기

## 여행지 첨부 흐름

```
여행지 검색
→ 검색 결과에서 추가
→ 선택 목록에 등록
→ 선택 순서대로 번호 부여
→ 위·아래 버튼으로 순서 변경
→ 지도에서 번호 마커와 직선 연결선 미리보기
```

드래그 앤 드롭은 제외하고 위·아래 버튼을 사용합니다.

## 모바일

```
[게시글 작성 패널]

[여행지 첨부 패널]

[취소] [등록]
```

좌우 패널을 세로로 전환합니다.

## 화면 구조

```
┌──────────────────────────────────────────────┐
│ 부산 하루 여행 코스                         │
│ 여행 코스 · 2026.07.14       [수정] [삭제]  │
├──────────────────────────────────────────────┤
│ 해운대부터 광안리까지 이동하는 코스입니다.  │
├──────────────────────────────────────────────┤
│ 여행 코스 지도                              │
│                                              │
│    ① 해운대 ───── ② 동백섬                 │
│                         │                    │
│                         └──── ③ 광안리      │
├──────────────────────────────────────────────┤
│ 코스 순서                                   │
│ 1. 해운대 해수욕장                          │
│ 2. 동백섬                                   │
│ 3. 광안리 해수욕장                          │
├──────────────────────────────────────────────┤
│ 선택 여행지 정보                            │
│ 해운대 해수욕장                             │
│ [지도에서 보기] [관련 게시글 보기]          │
├──────────────────────────────────────────────┤
│ 근처 여행지                                 │
│ 동백섬 · 달맞이길 · 블루라인파크            │
└──────────────────────────────────────────────┘
```

## 코스 지도

- 첨부 순서대로 번호 마커 표시
- 각 마커를 직선으로 연결
- 실제 도로 경로는 제공하지 않음
- 마커 클릭 시 해당 여행지 정보 표시

## 여행지 관련 이동

| 버튼             | 이동                      |
| ---------------- | ------------------------- |
| 지도에서 보기    | `/map?placeId=:placeId`   |
| 관련 게시글 보기 | `/posts?placeId=:placeId` |

## 수정·삭제

- 수정: `/posts/:postId/edit`
- 삭제: 비밀번호 확인 모달
- 비밀번호 불일치 시 삭제 차단

```

│                                  │                             │

│                                  │ · 동백섬                    │
│      ●             ●             │ · 광안리 해수욕장           │
│             ●                    │ · 달맞이길                  │

│          ●                       │                             │
```

- 전체 여행지 핀 표시

- 지도 확대·축소 완료
- 검색어 변경
- 카테고리 변경

# 10. 여행지 상세 Nested 패널

현재 지도 여행지 목록에서 한 곳을 선택하면 오른쪽 패널 내부가 상세 패널로 전환됩니다.

## 목록 상태

```
현재 지도에 보이는 여행지 24개

· 해운대 해수욕장
· 동백섬
· 광안리 해수욕장
· 달맞이길
```

## 상세 상태

```
┌──────────────────────────────┐
│ ← 여행지 목록                │
├──────────────────────────────┤
│ 해운대 해수욕장              │
│ 관광지 · 부산 해운대구       │
│                              │
│ 부산광역시 해운대구 우동     │
│                              │
│ [이 여행지로 코스 작성하기]  │
├──────────────────────────────┤
│ 관련 게시글                  │
│ · 부산 바다 따라 하루 여행   │
│ · 해운대에서 광안리까지      │
│ · 가족과 다녀온 부산 코스    │
│ [전체 보기]                  │
├──────────────────────────────┤
│ 근처 여행지                  │
│ · 동백섬             0.8km   │
│ · 달맞이길           1.6km   │
│ · 블루라인파크       1.9km   │
│                              │
│ 직선거리 기준                │
└──────────────────────────────┘
```

## 상세 패널 구성

- 여행지 이름
- 여행지 카테고리
- 주소 또는 위치
- 코스 작성 버튼
- 관련 게시글 최대 3개
- 근처 여행지 3~5개
- 목록 돌아가기 버튼

## 근처 여행지 선택

근처 여행지를 클릭하면 새로운 패널을 계속 중첩하지 않습니다.

```
해운대 상세
→ 동백섬 클릭
→ 지도 중심을 동백섬으로 이동
→ 동일한 상세 패널 내용을 동백섬으로 교체
```

# 11. 지도 검색 및 카테고리

## 검색

검색 대상은 다음으로 제한합니다.

- 여행지 이름
- 주소 또는 지역명

검색 결과 처리:

1. 조건에 맞는 핀만 표시
2. 오른쪽 목록 갱신
3. 결과가 한 개면 지도 중심 이동
4. 선택 시 상세 패널 표시

## 카테고리

```
[전체] [관광지] [맛집] [문화시설] [숙박] [쇼핑]
```

- 단일 카테고리 선택
- 선택된 카테고리 핀만 표시
- 오른쪽 목록에도 동일 조건 적용
- 전체 선택 시 초기화

## 필터 결합

```
검색어 + 카테고리 + 현재 지도 범위
```

세 조건을 모두 만족하는 여행지만 표시합니다.

# 12. 주변 여행지 추천

외부 API 없이 보유한 위도·경도로 계산합니다.

## 처리 과정

```
기준 여행지 선택
→ 다른 여행지와 직선거리 계산
→ 가까운 순 정렬
→ 자기 자신 제외
→ 상위 3~5개 반환
```

## 화면 표시

```
근처 여행지

동백섬             0.8km
달맞이길           1.6km
블루라인파크       1.9km

※ 직선거리 기준
```

# 15. 반응형 처리

| 화면        | 데스크톱                 | 모바일                                        |
| ----------- | ------------------------ | --------------------------------------------- |
| 게시글 작성 | 작성·첨부 패널 좌우 배치 | 세로 배치                                     |
| 여행지 지도 | 지도 + 오른쪽 패널       | 지도와 패널을 화면 전환 또는 하단 시트로 표시 |
| 여행지 상세 | 오른쪽 패널 내부 전환    | 전체 화면 또는 하단 시트                      |
| 게시글 목록 | 테이블형                 | 카드형 목록                                   |
| 챗봇        | 우측 하단 작은 대화창    | 전체 화면 채팅                                |

모바일 여행지도는 다음 방식이 적합합니다.

```
지도
→ 하단 여행지 목록 시트
→ 여행지 선택
→ 상세 하단 시트
```

# 17. 최종 UI 결정사항

| 영역             | 최종 결정                                  |
| ---------------- | ------------------------------------------ |
| 게시글 목록      | 제목 옆에 핀 아이콘과 `+N` 배지 표시       |
| 게시글 작성      | 일반 작성 패널과 여행지 첨부 패널 분리     |
| 여행지 첨부      | 검색, 카테고리, 다중 선택, 순서 변경 지원  |
| 코스 표시        | 번호 마커와 직선 연결선 사용               |
| 여행지 지도      | 지도와 현재 범위 여행지 목록을 함께 표시   |
| 지도 목록        | 현재 지도 화면에 보이는 여행지만 노출      |
| 여행지 상세      | 목록 패널 내부에서 Nested 상세 패널로 전환 |
| 상세 정보        | 기본 정보, 관련 게시글, 근처 여행지 표시   |
| 근처 여행지      | 좌표상 직선거리 기준 3~5개                 |
| 지도 검색        | 이름과 주소 검색                           |
| 지도 필터        | 카테고리 단일 선택                         |
| 챗봇             | 전역 플로팅 위젯                           |
| 실제 경로 안내   | 제외                                       |
| 여행 코스 시각화 | 포함                                       |
| 회원 기능        | 제외, 익명 구조 유지                       |

## 최종 핵심 경험

```
지도에서 여행지를 탐색한다.
→ 현재 화면에 보이는 여행지를 목록으로 확인한다.
→ 여행지를 선택해 상세·관련 게시글·근처 여행지를 확인한다.
→ 여러 여행지를 선택해 순서가 있는 여행 코스를 작성한다.
→ 게시글 상세에서 번호 마커와 연결선으로 코스를 확인한다.
```
